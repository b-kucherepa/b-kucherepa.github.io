Some of my works

C# & .NET
Testing tasks This small project accumulates the code I've written on my educational way. It represents different algorithms, such as sort, search or polynomial calculating ones, imbued with rich commentaries, also some handmade class implementations and task solutions, which are combined together in console menu.
Every implementation is provided with simple but demonstrable testing interfaces.		 Link to the git repository
Nightwrap A minimalistic Windows desktop service which conceals the screen with black screensaver in a specified time interval if was no input detected, without putting the PC into sleep.
Supports specifying the interval, working in the system tray, and loading on Windows startup. 		Link to the git repository
ReaderX A Windows desktop app designed to display text over changeable background images which are dynamically loaded from the specified folder. Thus it combines reader and slideshow functions.
Also provides font formatting, installing random or consequential order of slideshow with the adjustable interval, selecting specific background from the list, and usage of hotkeys.		 Link to the git repository
