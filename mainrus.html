Nightwrap Минималистичная десктопная служба для Windows, которое скрывает экран c помощью черного скринсейвера через установленный интервал времени, если не было зафиксировало ввода, не вводя ПК в режим сна.
Поддерживает установку интервала, работу в системном трее, и автозагрузку при запуске системы. 		Ссылка на репозиторий git
ReaderX Декстопное приложения для Windows, разработанное для отображения текста поверх сменяемых фоновых изображений,которые динамически загружаются из указанной папки. Таким образом, приложение объединяет функции "читалки" и слайдшоу.
Также позволяет менять форматирование текста, устанавливать случайный или последовательный порядок слайдшоу с настраиваемым интервалом, выбирать определенный фон из списка, и использовать горячие клавиши.		 сылка на репозиторий git
